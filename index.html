URL API

Cara mendapatkan URL ini: buka Google Sheets Anda → menu Extensions > Apps Script → tempel kode dari file Code_GoogleAppsScript.gs → Deploy > New deployment (tipe Web app, akses "Anyone") → salin URL yang muncul lalu tempel di kolom atas.
